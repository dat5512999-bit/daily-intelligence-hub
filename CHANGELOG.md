# Changelog

## 0.9.1 - 2026-07-10

- 新增 GitHub Actions 每日自動產生報告與 GitHub Pages 發布設定。
- 產生 `reports/index.html`，可直接作為手機網頁首頁。
- 新增繁體中文的 GitHub Pages 手機使用說明。

## 0.9.0 - 2026-07-10

- 將手機預覽改為深色「個人情報控制台」：任務式必看情報、可辨識的頻道色彩與控制台閱讀流程。
- 保留來源數量、白話重點、深入閱讀與可信度提示，避免只剩視覺效果。

## 0.8.0 - 2026-07-10

- 重做個人化手機介面：深藍主視覺、今日必看三件事與 AI、遊戲、動漫、台中、股票、活動分區。
- 移除泛用卡片牆式閱讀，改為依「跟你有什麼關係」與來源數量呈現。

## 0.7.0 - 2026-07-10

- 新增個人興趣設定，依 AI／Codex、遊戲與電競、動漫、台中即時與活動、股票市場重新蒐集及過濾內容。
- 新增針對個人興趣的免費 Google News RSS 查詢；不再以泛用熱門話題主導報告。

## 0.6.0 - 2026-07-10

- 依使用者實例改為「這件事跟我有什麼關係」優先：辨識科技大事、市場變化、限時好康與在地突發事件。
- 移除無意義的內部重要性分數，改用可讀的來源與關聯說明。
- 離線 Demo 明確標示為範例，避免被誤解成即時趨勢。

## 0.5.0 - 2026-07-10

- 依實際使用回饋加入白話重點、可展開的出現原因，以及清楚的原文／繁中深入閱讀入口。

## 0.4.0 - 2026-07-10

- 依懶人使用者旅程調整：一鍵啟動改為即時模式並自動開啟報告。
- 報告加入「今天只看這三件事」、延後閱讀區與無資料時的明確提示。
- 新增「怎麼避免假消息」說明，避免誤將社群熱度當成事實。

## 0.3.0 - 2026-07-10

- 轉型為免費台灣生活趨勢雷達，新增 Dcard、Google 熱門搜尋與 TikTok 公開趨勢來源。
- 報告新增跨平台與單一社群熱度標示，避免將流行話題當作查證事實。

## 0.2.0 - 2026-07-10

- 加入生活情報分類：旅遊、美食、健康、娛樂與全球重要事件。
- 擴充繁中 Google News 類別與生活類 Reddit 來源。
- 加入可自行設定的公開 YouTube 頻道資料來源。

## 0.1.0 - 2026-07-10

- 建立 Sprint 1 MVP：Demo/Live、四來源 plugin、篩選、跨來源排名、Markdown 與 HTML 預覽。
- 新增啟動腳本、測試與完整基礎操作文件。
