# 手機查看：GitHub Pages

這個專案已準備好每小時自動產生網頁版報告。

## 啟用後會發生什麼事

- GitHub 會約每小時自動執行一次 `python main.py --live`。
- 它會抓取公開來源、產生最新情報，並把手機網頁更新到 GitHub Pages。
- 手機只要開啟同一個網址，就能閱讀最新內容；可將它加入手機桌面。

## 第一次啟用（GitHub 專案建立後）

1. 在 GitHub 專案頁面按上方的 `Settings`。
2. 左側找到 `Pages`。
3. 在 `Build and deployment` 的 `Source` 選擇 `GitHub Actions`。
4. 到 `Actions` 頁面，點選「每日情報更新」。
5. 點選 `Run workflow`，再按一次綠色按鈕手動執行第一份報告。
6. 執行完成後，進入 `Settings → Pages`，就能看到你的手機網址。

## 手機使用

在手機瀏覽器打開網址後：

- iPhone Safari：按分享按鈕，選「加入主畫面」。
- Android Chrome：按右上角選單，選「加到主畫面」。

之後桌面上的圖示就像一般 App；點開即可看最新情報。

## 注意事項

- GitHub 的免費排程偶爾可能延遲，不保證剛好在每個整點更新。
- 報告只會使用公開資料來源；某些來源暫時無法讀取時，當天報告仍會生成，但內容可能較少。
- 公開專案和 GitHub Pages 的網址，任何拿到網址的人都能查看。請不要放入私人帳號、密碼或個人資料。
