# 使用者操作手冊

1. 直接雙擊 `start.bat`；它會優先使用系統 Python，否則使用 Codex 已附帶的 Python 3.12，並自動開啟報告。
2. 先看上方數字與「效率掃描」，再用分類快速入口跳到你想看的生活流行、遊戲、動漫、股票、台中等頻道。
3. 若只想檢查畫面，雙擊 `start_demo.bat`。

## 先看「資料狀態」再閱讀

- `來源正常`：至少三個來源成功，且本次沒有來源錯誤。
- `部分來源受限`：仍有至少三個來源完成，但例如 Dcard、Reddit 等其中一個平台暫時限制讀取；內容可看，但社群面向可能較少。
- `資訊來源不足`：只取得一到兩個來源，請先當速報看，下一小時會再自動更新。
- `資料不足`：本次沒有足夠即時資料；系統不會用範例內容冒充真實資訊。

## 看不懂或想深入怎麼辦？

- 先看每張卡片的「白話說」。
- 不懂為何出現，點開「為什麼會出現在這裡？」。
- 對某件事有興趣，按「看原文」。
- 原文是英文或其他語言，按「翻成繁中閱讀」，會在 Google 翻譯開啟該頁面。
- 如果某分類暫時不想看，按「少看這類」；如果想多看，按「多看這類」。目前回饋會先記在同一台手機或瀏覽器。
3. 若要使用即時模式，開啟終端機至專案根目錄後，使用 `python main.py --live`；沒有系統 Python 時，請改用 start.bat 內的 Codex Python 路徑。

## 加入想看的 YouTube 頻道

開啟 `config/youtube_channels.json`，將範例改成頻道資料，例如：

```json
[
  {"name": "我喜歡的旅遊頻道", "channel_id": "頻道 ID", "category": "旅遊生活"}
]
```

可使用分類：`影音生活`、`旅遊生活`、`美食生活`、`健康生活`、`娛樂文化`、`科技`。Sprint 2 的 YouTube 功能只追蹤你選定的公開頻道，保護隱私也避免大量雜訊。

若某來源無法使用，報告仍會建立，並在終端機與報告中標示來源警示。

## 個人化內容

目前預設的主題為：AI／Codex、生活流行、遊戲與電競、動漫、台中即時事件、台中好康活動、股票與市場。生活流行會參考 Dcard 公開熱門討論與小紅書相關公開搜尋/新聞訊號；遊戲包含幻獸帕魯、GTA、SF Online、地平線/Forza、魔獸爭霸/Warcraft 等。這些設定存放在 `config/interests.json`，不需要帳號、資料庫或 API Key；日後只要告訴 Codex 想增加或減少哪些主題即可。
