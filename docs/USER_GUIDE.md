# 使用者操作手冊

1. 直接雙擊 `start.bat`；它會優先使用系統 Python，否則使用 Codex 已附帶的 Python 3.12，並自動開啟報告。

新版首頁不會讓每一則情報都長得一樣：最上方 Hero 是今天最值得先看的事；紅色 Alert 是需要注意的突發狀況；排行榜表示搜尋或社群熱度；有可用官方縮圖的 YouTube 才會有圖片卡；股票、AI、遊戲、動漫、台中與生活資訊使用短清單與快速入口呈現。

0.9.19 起，畫面頂端有「AI／Codex、遊戲與電競、動漫與娛樂、你的持股雷達、台中、生活流行、GitHub」中文快速入口。每一個頻道最多三則；如果當輪沒有抓到可靠內容，會直接顯示原因。未翻譯的英文 Reddit／Hacker News 貼文與 RSS 內的網頁程式碼不會顯示在首頁。

0.9.20 起，首頁只先顯示三件必看情報。要看其他類別時，點頂端分類，該頻道才會展開。「👍 有興趣」與「🙈 少一點」會存在這支手機的瀏覽器中；不需要帳號，但換手機或清除瀏覽器資料後會重設。

0.9.21 起，頁首有兩個更新工具：「檢查最新頁面」只會重新載入已部署的內容；「手動產生新報告」會開啟 GitHub Actions。若要立刻抓新資料，請按後者，再於 GitHub 按 **Run workflow**。公開的 GitHub Pages 不能安全保存可代替你操作 GitHub 的權杖，所以不會設計成網頁直接偷偷執行。

0.9.22 起，頂端新增「年輕人現在紅什麼」與「運動與 NBA」。年輕人頻道會找搶購商品、溝通現象、迷因與國內外網美景點；運動頻道會找 NBA 球員交易、台灣賽事、足球與挪威足球。各頻道採三輪補位，候選足夠時固定顯示 3 則；若公開來源真的失敗，仍會誠實顯示不足，不用假資料湊數。
2. 先看上方數字與「效率掃描」，再用分類快速入口跳到你想看的生活流行、遊戲、動漫、股票、台中等頻道。
3. 若只想檢查畫面，雙擊 `start_demo.bat`。

首頁前三張是今日精選；往下滑的分類只會顯示其他新內容，不會把同一篇文章重複塞給你。

卡片上的第一個標籤是閱讀建議：`現在看` 代表可能影響今天行程或你的持股；`值得看` 建議花 1 分鐘掌握；`有空看` 偏生活安排；`路過知道即可` 則不需要立刻點開。卡片底部會說明它屬於官方公開資訊、媒體報導、搜尋熱度或社群討論。

## 先看「資料狀態」再閱讀

- `來源正常`：至少三個來源成功，且本次沒有來源錯誤。
- `部分來源受限`：仍有至少三個來源完成，但例如 Dcard、Reddit 等其中一個平台暫時限制讀取；內容可看，但社群面向可能較少。
- `資訊來源不足`：只取得一到兩個來源，請先當速報看，下一小時會再自動更新。
- `資料不足`：本次沒有足夠即時資料；系統不會用範例內容冒充真實資訊。

## 看不懂或想深入怎麼辦？

- 先看每張卡片的「白話說」。
- 不懂為何出現，點開「為什麼會出現在這裡？」。
- 對某件事有興趣，按「看原文」。
- 原文是英文或其他語言，按「翻成繁中閱讀」，會在 Google 翻譯開啟該頁面。
- 對某張卡片沒興趣，按「少看這則」；想留意某張卡片，按「多看這則」。只會影響該張卡，不會連動同分類其他內容；目前回饋會先記在同一台手機或瀏覽器。
3. 若要使用即時模式，開啟終端機至專案根目錄後，使用 `python main.py --live`；沒有系統 Python 時，請改用 start.bat 內的 Codex Python 路徑。

## 加入想看的 YouTube 頻道

開啟 `config/youtube_channels.json`，將範例改成頻道資料，例如：

```json
[
  {"name": "我喜歡的旅遊頻道", "channel_id": "頻道 ID", "category": "旅遊生活"}
]
```

可使用分類：`影音生活`、`旅遊生活`、`美食生活`、`健康生活`、`娛樂文化`、`科技`。Sprint 2 的 YouTube 功能只追蹤你選定的公開頻道，保護隱私也避免大量雜訊。

若某來源無法使用，報告仍會建立，並在終端機與報告中標示來源警示。

## 個人化內容

目前預設的主題為：AI／Codex、年輕人流行、生活流行、遊戲與電競、動漫、運動與 NBA、台中即時事件、台中好康活動、股票與市場及 GitHub。生活流行會參考 Dcard 公開熱門討論與小紅書相關公開搜尋／新聞訊號；遊戲包含幻獸帕魯、GTA、SF Online、地平線／Forza、魔獸爭霸／Warcraft 等；運動包含 NBA 交易、台灣賽事與挪威足球。這些設定存放在 `config/interests.json`，不需要帳號、資料庫或 API Key；日後只要告訴 Codex 想增加或減少哪些主題即可。
